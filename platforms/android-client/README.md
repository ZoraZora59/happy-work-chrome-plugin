# 安卓客户端（待实现）

本目录为安卓客户端的实现占位，建议使用 Kotlin/Jetpack Compose 完成界面与逻辑开发，并尽量复用浏览器插件的配置与计薪算法。

## 建议结构
```
platforms/android-client/
├── app/
│   ├── src/main/java/...   # 业务逻辑与页面
│   ├── src/main/res/       # 资源文件
│   └── build.gradle        # 模块配置
└── build.gradle            # 工程配置
```

## 开发提示
- 可将浏览器插件中的收入计算与心情状态逻辑提取为共享模块，按需迁移到 Kotlin。
- UI 建议保留实时收入、进度条、价值换算等核心展示，交互与浏览器端保持一致。
- 重要配置（薪资、工作时间、加班规则等）可存储在本地数据库或 DataStore，保证重启后数据持久化。
